Borders are boring. They sit there, static, doing the minimum. But give that border a moving current of light and suddenly your card feels like it's powered on, ready to react. That's what the Electric Border component does.

## What it does

A glowing light pulse travels continuously along the border of any container. The effect looks like electrical current moving through a wire, completing laps around the element indefinitely.

## How it works

The animation uses a pseudo-element technique with a conic-gradient that rotates around the element's border. A CSS custom property tracks the angle, and a keyframe animation increments it from 0 to 360 degrees on a loop.

The glow is layered with a blur and a color tint, giving it that plasma-like quality. The border itself stays transparent so only the animated gradient shows through.

## Key properties

- **color** - The primary glow color (default: electric blue)
- **duration** - Time in seconds for one full lap (default: 2s)
- **borderWidth** - Thickness of the glowing border (default: 2px)
- **glowSize** - How far the glow bleeds beyond the border
- **borderRadius** - Matches rounded corners on the container

## When to use it

Use this on feature cards, pricing plan highlights, or any container you want to signal as active or premium. It draws the eye without requiring the user to hover.

Keep the duration between 1.5s and 3s. Faster feels jittery, slower loses the energy that makes it interesting.
